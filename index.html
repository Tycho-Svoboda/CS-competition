Python

Python is a coding language that is very popular right now, and is being used for new technologies such as machine learning neural networks. It’s an interpreted language, meaning that it is translated to the machine during the execution of the program. Luckily for you, it is easy to learn and has basic syntax. We’ll be going over some basic coding concepts that are universal in every language, and go into the syntax python uses to represent those ideas.

Variables:

Variables are used to store data, and take up memory from the computer. Each variable has a name, and python will match the type (i.e integers, strings/words, etc.) The syntax, or way that you type the code so Python understands is:

(varName) = (Data)

Super simple, right?

Let’s say you want to make whoever is looking at your code to have a better understanding of what your variable’s purpose is. To do this, you’d insert a comment, which is anything following a double slash as seen below.

//This is a comment, the interpreter won’t use this while it runs

When you mix comments with variables, your code will be easily understandable by anyone. In coding, it’s important that your code is understandable and reads out quickly to people not familiar for what you’re working on. For example, take a look at the following code, it tells you important information about the variable.

//This variable is used to store the price of a gallon of milk
galMilkPrice = 2.20

Variables should always be in camel case, which essentially means that each new word after the first word is upper case, with the first word being lowercase. So instead of Gal Milk Price to represent milk gallon price, you would use galMilkPrice.

Python will automatically assign a data type to the variable, based on what data you put in. For example, in the code above, galMilkPrice will be stored as a float (integer with decimal point), to match 2.20.

Here are some important data types you should be familiar with:

float - a decimal number

bool - boolean, true or false value

int - integer, a non decimal number

Machine Learning

Introduction

Machine learning is a method of data analysis that automates analytical model building. It is a branch of artificial intelligence based on the idea that systems can learn from data, identify patterns and make decisions with minimal human intervention.

Applications of Machine Learning:

  • Amazon and Netflix recommendations
  • Fraud detection
  • Twitter, Facebook, Instagram, etc.

It enables computers to quickly and automatically produce models that can analyze bigger, more complex data and deliver faster, more accurate results - even on a very large scale. By building precise models, and organization has a better chance of identifying profitable opportunities - or avoiding unknown risks.

Machine Learning Methods

Supervised Learning
  • Algorithms are trained using labeled examples, including an input where the desired output is known.
  • The learning algorithm receives a set of inputs along with the corresponding correct outputs, and it learns by comparing the calculated outputs with the correct outputs and identifying the errors.
  • It uses methods such as classification, regression, prediction and gradient boosting to predict values on additional unlabeled data.
Unsupervised Learning
  • Used against data that has no historical labels - the system is not given the correct output.
  • It learns and understands what is being given. The goal of the algorithm is to find a pattern or structure within the data.
Semi-supervised learning
  • Used for same applications as supervised learning
  • Uses both labeled and unlabeled data for training
  • Used when cost associated with labeling is too high for a fully labeled training process.
Reinforcement Learning
  • With reinforcement learning, the algorithm discovers through trial and error the most profitable actions
  • Three primary components
    • Agent (learner)
    • Environment (what the agent interacts with)
    • Actions (what the agent can do)
  • Objective is for the agent to choose actions that maximize expected reward over a given amount of time.
